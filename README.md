## Run Locally

Clone the MacBookAirM2

```bash
  git clone https://github.com/EdgarHdzHdz17/MacBookAirM2.git
```

Go to the project directory

```bash
  cd MacBookAirM2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
